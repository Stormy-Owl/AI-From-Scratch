# AI-From-Scratch
AI From Scratch is a collection of machine learning and deep learning algorithms implemented entirely in pure Python from first principles, with every core component built manually to understand the inner workings of artificial intelligence.

Features
🧠 Neural Networks from scratch
📷 Convolutional Neural Networks (CNN)
🔄 Recurrent Neural Networks (RNN)
📐 Forward & Backpropagation
📊 Gradient Descent and Optimization
